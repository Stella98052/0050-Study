## 一、ZigZag 轉折偵測（zigzag.py）
═══════════════════════════════════════════
```
確認規則（threshold 預設 5%，Config.zigzag_threshold）：
  峰確認 = 當日 low  ≤ 波段最高 high × (1 − threshold)
  谷確認 = 當日 high ≥ 波段最低 low  × (1 + threshold)
confirmed_date = 達到反向幅度的「當日」（必晚於轉折日 = realtime 延遲來源）

retrospective 版：全部轉折 + 最後暫定極值，confirmed_date=None ◇僅視覺化
realtime 版：僅回傳 confirmed_date ≤ as_of 的轉折 ★可入訓練
閾值影響：越小→轉折多/雜訊高；越大→平滑/確認延遲久
```

## 二、三大鐵律（wave_labels.py，明確布林判斷式）
═══════════════════════════════════════════
```
鐵律① wave2_not_break_wave1_origin：第2浪低點 > 第1浪起點
鐵律② wave3_not_shortest：
   波5未現（realtime常態）→ len3 ≥ len1，key='wave3_not_shortest_partial'（暫定）
   波5已現 → NOT(len3 < len1 AND len3 < len5)，key='wave3_not_shortest_final'
鐵律③ wave4_not_overlap_wave1：第4浪低點 > 第1浪高點
違反任一 → 該段標 unknown 並以違規谷重啟循環計數
```

### 推進/修正浪量化判別（approximation）
```
impulse：順主趨勢 AND 幅度 ≥ 相鄰逆勢段 × impulse_min_amplitude_ratio(1.0)
         AND K棒數 ≥ impulse_min_duration_bars(2)
corrective：逆主趨勢 AND K棒數 ≥ 門檻
其餘 → unknown（不硬判）
斐波那契回撤 0.382/0.5/0.618 ± 0.05 僅輔助標記，非判定依據
```

### realtime 逐日標籤
```
當日標籤 = 最後一段已完成波之「下一序」（例：波2谷確認 → 當日 '3'）
逐列保證 label_basis_last_confirmed_date ≤ date（pytest 斷言）
```

## 三、MV 潮汐量能（volume_features.py，定案2：5/13/20 三線）
═══════════════════════════════════════════
```
mv_k = volume.rolling(k).mean()        k = 5（攻擊）/ 13（核心否決）/ 20（乖離分母）
方向_k = sign(mv_k(今) − mv_k(昨))      +1↑ / −1↓ / 0持平
量能乖離率 mv_bias = mv_short / mv_long − 1
潮汐爆發 is_volume_burst = mv_bias > 0.20
13MV核心否決 mv_mid_veto_active = (mv_mid_direction < 0)   ← 方法論絕對否決
量價背離 = realtime波3 AND close > 前日以前歷史最高close AND 5MV方向 ≤ 0
          （新高基準用 expanding().max().shift(1)，防未來函數）
```

## 五、VG-1 資料完整性（vg1.py + trading_calendar.py）
═══════════════════════════════════════════
```
理論交易日曆 = 同批股票實際交易日「聯集」（approximation，定案1）
完整率 = 該股實際交易日數 ÷ 日曆在該股期間內日數；< 0.95 → 未通過+列缺漏區間
檢查：dtype / 重複日期 / 價量負值 / 缺漏區間（相隔>7日切段）
輔助校驗：僅於官方休市日表涵蓋期間 [min,max] 內，標記
   「平日 ∧ 非官方休市 ∧ 不在聯集」= 疑似缺漏；範圍外不校驗
已知限制：全部股票同日皆缺漏→無法偵測；臨時休市（颱風）不在年度表
   → 需人工查證（案例：2026-07-10 巴威颱風休市，經新聞查證為真休市）
```

### 標籤（定案：淨報酬>0）
```
entry = 訊號日下一交易日開盤 open[t+1]
exit  = 第N交易日收盤 close[t+N]（N=5）
fwd_return_gross = exit/entry − 1
fwd_return_net   = gross − (0.1425% + 0.1425% + 0.3%)   成本以報酬扣減近似
label_up = fwd_return_net > 0；末N列無未來窗 → NaN 剔除
```

### 特徵（FEATURE_COLS，全部僅用 t 以前資料）
```
波浪 wave_label_realtime（one-hot 固定9類）｜MV 5/13/20 值+方向+bias+burst+veto
RSI14 = Wilder：100−100/(1+平均漲幅/平均跌幅)，ewm alpha=1/14
MACD = EMA12−EMA26；訊號線 EMA9；柱=差
過去報酬 ret_5d/ret_20d = close.pct_change(5/20)
```

### Walk-Forward
```
訓練3年｜測試3月｜步長3月｜Embargo 30天（≥MACD慢線26）
holdout = 最末12個月，完全不進任何折與敏感度搜尋（最終樣本外）
⛔每折斷言 train_end + embargo < test_start
```

### 績效指標（gross/net 並列，v2.3 定案）
```
交易層（全部交易）：勝率 = 獲利筆數/總筆數；盈虧比 = 平均獲利/|平均虧損|
                    avg_trade_net = 單筆平均淨報酬
路徑層（僅不重疊交易，間隔≥N 同 VG-4 規則、共用 select_independent_dates）：
  總報酬/MDD = 不重疊交易原始報酬直接連乘之淨值曲線（真實可執行單部位路徑）
  Sharpe = mean(獨立報酬)/std × √(252/N)
  Alpha = 獨立路徑總報酬 − 同期0050買進持有
  n_independent < VG-4門檻 → 路徑層數字不具統計意義，VG-4警語蓋過
禁止：重疊觀測序列鏈乘（v2.0）、日籃鏈乘（v2.1）、日等效鏈乘（v2.2）
  —— 詳見 LESSONLEARNT L9/L10
```

### VG-3 統計顯著性（v2.4 定案）
```
輸入（強制）：signal_returns = independent_return_series 篩出之統計獨立子集
  （同日等權合併 → 間隔≥N 日曆日 greedy；與 VG-4/權益曲線共用同一核心）
  禁止餵入未篩選交易層序列 —— 重疊觀測=偽重複，人為壓低 p / 收窄 CI（L13）
Permutation：自全體報酬池抽 len(獨立子集) 樣本×500次 → p = P(隨機均值 ≥ 真實均值)
Bootstrap：對獨立子集重抽樣1000次 → 95% CI；CI含0 = 無法證明優於隨機
蓋過規則：VG-4 未達門檻 → VG-3 結果標「未決(VG-4蓋過)」，不得引用
```

### 持有期研究基準（v2.4，L14）
```
baseline_N = 同池同期「任一日進場、持有N日」之平均淨報酬（市場飄移）
edge_N = 訊號獨立均值 − baseline_N
判讀：edge 隨 N 擴大才支持成本攤薄假說；淨報酬絕對值由市場環境決定
```

### VG-5 防洩漏（程式化斷言，非文件聲明）
```
①欄名黑名單：任何 retrospective 欄 → AssertionError
②截斷重算：抽3個時點t，僅用[:t+1]重算特徵，與全量第t列逐欄一致
③結構斷言：末N列標籤必為NaN
④切分斷言：train_end+embargo < test_start
```

### VG-6 模型輸出健康度（v2.9）
```
檢查（逐列，非多股聚合）：std<0.02=近常數｜[0.45,0.55]占比>90%=無判別帶
  ｜選中率>95%或<5%=系統性單邊｜AUC<0.55=無判別力（Mann–Whitney 平均秩法）
AUC = (正類秩和 − n₊(n₊+1)/2) / (n₊·n₋)
防誤判註記：多股池「任一檔/日」聚合占比 1−(1−p_row)^k 天然趨近 100%，
不得作為退化證據；一切以逐列分布與 AUC 為準（L19）
```
